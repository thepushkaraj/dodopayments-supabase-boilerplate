üß™ End-to-End Testing Flow

Follow these steps to test the complete integration:

  1. Configure URLs: Ensure the Supabase Functions Base URL and Anon Key are set correctly above.
  2. Create a Customer: Use Step 1 to sign up a new user. This creates records in both Supabase and Dodo Payments.
  3. Create Subscription: Use Step 2 to initiate a subscription for the customer from Step 1.
  4. Complete Payment: You will be redirected to the Dodo Payments checkout page. Complete the test payment.
  5. Check Webhook Logs: In your terminal, view the webhook logs using `npx supabase functions logs webhook`. You should see a `payment.succeeded` and `subscription.active` event.
  6. Verify Database: Go to your Supabase dashboard and check that the `subscriptions` table for your customer has been updated to `active`.
